# Foglio.md 📝

**Foglio.md** is a modern, minimal, and open-source Markdown editor designed to make writing fluid and distraction-free. Built with **React** and **Vite**, **Docker**, **MongoDB**, **NestJS** and **Firebase**, it offers an intuitive interface for those who want the power of Markdown with the convenience of an immediate, glassmorphic toolbar.

## ✨ Features

- **Pure Minimalism:** A clean writing area focused entirely on your content.
- **Intuitive Toolbar:** Quick support for standard formatting (Bold, Italic, Headers).
- **Advanced Elements:** Easy management of tables, checklists, code blocks, and links.
- **Native Dark Mode:** An elegant dark design to reduce eye strain during long writing sessions.
- **High Performance:** Lightning-fast experience powered by the Vite build tool.

## 🚀 Getting Started

To run the project locally, follow these steps:

### 1. Clone the repository
```bash
git clone [https://github.com/corrado-i/fogliomd.git](https://github.com/corrado-i/fogliomd.git)

```
### 2. Create project in Firebase

1. Go to Firebase Console
2. Click on "Add project"
3. Enter project name
4. Click on "Continue"
5. Click on "Create project"
6. Disable Google Analytics
7. Click on "Create project"
8. Go to project settings
9. Click on "Add app"
10. Click on "Web"
11. Enter app name
12. Click on "Register app"
13. Copy the Firebase configuration

### 3. Copy .env-example to .env and paste the Firebase configuration in the .env file

```bash
cd fogliomd
cp .env-example .env
```

### 4. Run the application

```bash
cd fogliomd
docker compose up -d --build

```
